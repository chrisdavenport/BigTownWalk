Dashboard

  • Walk. An individual walk event that takes place at a particular time with a particular leader along a particular route.
  • Route A complete walk follows a route comprising one or more consecutive segments joined by locations.
  • Segment A section of a route connecting a pair of locations. Each segment comprises a start location and a finish location, together with a set of intermediate waypoints.
  • Location A point along a route that has some significance to walkers. It might be a starting or finishing point, or a point of interest, for example.
  • Waypoint A point along a route. Collectively, all the waypoints in a route define that route. A location is essentially just a special kind of waypoint that has some additional information associated with it.
  • Leader The name and possibly other information concerning the person who is leading a walk.
